# SMARTMAINTAIN

Production-ready Maintenance Management System built with Next.js 15, Tailwind CSS, and Supabase.

## Features
- **Multi-Role Authentication**: Tenants, Technicians, Managers, and Admins.
- **Maintenance Requests**: Submit, track, assign, and resolve.
- **Property Management**: Manage properties and individual units.
- **Image Attachments**: Attach photos to maintenance requests.
- **Secure Access**: Row-Level Security (RLS) ensures tenant isolation.
