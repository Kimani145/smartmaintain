# Roadmap

## Post v1.0 Release Candidate

### Immediate (v1.1)
- Implement comprehensive notification center UI for all roles.
- Add full reporting and analytics dashboard for Admins.
- Add real-time WebSocket subscriptions for maintenance request status changes.
- Add unit tests (Vitest) and e2e tests (Playwright) to the CI/CD pipeline.

### Short-term (v1.2)
- Advanced filtering and pagination for Manager data tables.
- Batch tenant import via CSV.
- Mobile-native styling optimizations (PWA capabilities).

### Long-term (v2.0)
- Payment Gateway Integration for automated rent collection.
- Vendor portal for external contractors.
- Automated lease generation and digital signatures.
